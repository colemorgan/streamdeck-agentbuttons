# Stream Deck profiles

Prebuilt layouts for Codex Agent Buttons.

## Manual layout (all models)

Until exported `.streamDeckProfile` files are checked in for your Stream Deck Software version:

1. Create a new profile (or page).
2. Add six **Agent Slot** actions in a row or 2×3 block.
3. Set slots left→right (or reading order) to **1 through 6**.
4. Optional display names: match your usual agents.

### Stream Deck XL suggestion

Place A1–A6 on the bottom row or a dedicated agent column so they stay visible while other actions fill the rest of the deck.

### Standard / MK.2

Use a full page of six agent keys, or keep three “primary” agents on the home page.

### Neo / smaller decks

Map only the slots you use (e.g. 1–3). Logical slots 4–6 still exist in ChatGPT; they just need no physical key.

## Export (maintainers)

After laying out keys in Stream Deck Software, export the profile and add it here as:

- `xl-six-agents.streamDeckProfile`
- `standard-six-agents.streamDeckProfile`

Update this README with import steps for the current Stream Deck Software version.
