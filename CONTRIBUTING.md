# Contributing

Personal / open-source project. PRs welcome for bugfixes and packaging improvements.

## Develop

```bash
npm install
npm test
npm run build
```

- Plugin: `npm run plugin:build` then `streamdeck restart com.colemorgan.codex-agent-buttons`
- Companion CLI: `node companion/dist/cli.js --verbose --chatgpt`
- Companion app: `npm run companion:app`
- Release pack: `npm run plugin:pack`
